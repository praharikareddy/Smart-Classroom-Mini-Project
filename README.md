# 📚 Digital Twin Smart Classroom System

## 📌 Project Overview
The Digital Twin Smart Classroom System is a 3D-based simulation project that creates a virtual replica of a real classroom environment. This system uses Blender and Python scripting to simulate environmental conditions such as temperature and student occupancy, and automatically controls classroom devices like fans and lights.

The project demonstrates how digital twin technology can be applied in smart education systems to improve energy efficiency, comfort, and automation.

---

## 🎯 Objectives
- To design a 3D virtual classroom using Blender  
- To simulate real-time classroom conditions (temperature & occupancy)  
- To automate control of fans and lights based on conditions  
- To visualize system behavior dynamically  

---

## ⚙️ Features
- 🌡️ Temperature simulation  
- 👥 Student occupancy simulation  
- 🌀 Automatic fan rotation based on temperature  
- 💡 Smart lighting control based on occupancy  
- 🖥️ Real-time status display (Temp, Students, Devices)  
- 🎬 Smooth animation using Blender  

---

## 🛠️ Technologies Used
- **Blender** – 3D Modeling & Visualization  
- **Python (bpy)** – Automation & Control Logic  

---

## 🧠 Working Principle
The system generates random values for temperature and student count to simulate real-world conditions. Based on these values:
- Fans turn ON when temperature exceeds a threshold  
- Lights turn ON when students are present  
- The classroom board displays real-time updates  

All changes are visualized instantly in the 3D environment.

---

## ▶️ How to Run the Project
1. Open Blender  
2. Load the file: `smart_classroom.blend`  
3. Open the scripting workspace  
4. Run the Python script  
5. Observe real-time simulation and animation
